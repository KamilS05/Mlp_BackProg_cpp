# Multilayer Perceptron in C++ (Backpropagation)

This repository contains two implementations of a Multilayer Perceptron (MLP) written in C++.

The projects were developed for educational puropses focused on understanding neural networks and backpropagation from scratch.

---

### la1

Basic implementation of a Multilayer Perceptron trained using online backpropagation with momentum.

Focused on:
- Forward propagation
- Backpropagation
- Mean Squared Error (MSE)
- Simple regression / small datasets (e.g., XOR)

---

### la2

Extended version oriented toward classification problems.

Includes:
- Online and offline training
- Softmax output layer
- Cross-Entropy loss
- Classification accuracy (CCR)
- Confusion matrix
- Kaggle-style prediction mode
- Multiple random seeds evaluation

---

Each folder contains its own:
- Source files
- Makefile
- Dataset examples
- Detailed README with build and usage instructions
